## 进行中 ⏳

### Phase 2: 深度内容分析（当前重点）

#### Step 2.1: 视频风格分析
**目标**：分析"第四种黑猩猩"的独特风格特征

**分析维度**：
- [ ] 视频节奏（语速、停顿、剪辑风格）
- [ ] 视觉呈现（封面、配图、字幕样式）
- [ ] 语言风格（专业度、口语化、幽默感）
- [ ] 个人特色（口头禅、标志性表达）

**输出**：`style_analysis.md`

#### Step 2.2: 选题策略分析
**目标**：分析爆款视频的选题规律

**分析维度**：
- [ ] 选题类型分布（热点/教程/观点/故事）
- [ ] 选题时机（跟热点速度、首发优势）
- [ ] 选题角度（差异化切入、独特观点）
- [ ] 选题系列化（主题连贯、成体系）

**输出**：`topic_strategy.md`

#### Step 2.3: 内容结构拆解
**目标**：深度拆解视频的内容架构

**分析维度**：
- [ ] 开头设计（前3秒钩子、开场白）
- [ ] 主体展开（信息密度、逻辑层次）
- [ ] 结尾处理（总结、升华、CTA）
- [ ] 节奏控制（高潮设置、情绪曲线）

**输出**：`structure_breakdown.md`

#### Step 2.4: 互动策略分析
**目标**：分析如何提高互动率

**分析维度**：
- [ ] 评论引导（提问、争议、共鸣）
- [ ] 点赞激励（内容价值、情感共鸣）
- [ ] 收藏理由（实用干货、长期价值）
- [ ] 分享触发（社交货币、身份认同）

**输出**：`engagement_strategy.md`

## 待开始 ⏸️

### Phase 3: 对标文档生成

#### Step 3.1: 综合对标文档
**目标**：整合所有分析，生成完整对标文档

**文档结构**：
1. **博主画像**：风格定位、人设特点、目标受众
2. **内容策略**：选题方向、内容类型、发布节奏
3. **创作方法**：标题公式、结构模板、话术库
4. **运营技巧**：互动策略、增长方法、变现路径
5. **差异化建议**：如何在学习基础上形成个人特色

**输出**：`benchmark_report.md`（主文档）

#### Step 3.2: 可复用模板库
**目标**：提取可直接使用的模板

**模板类型**：
- [ ] 标题公式库（10+公式）
- [ ] 开场白话术库（20+话术）
- [ ] 内容结构模板（5+结构）
- [ ] 结尾CTA话术库（30+话术）
- [ ] 评论区回复模板（20+回复）

**输出**：`template_library.md`

#### Step 3.3: 行动清单
**目标**：提供可执行的启动方案

**清单内容**：
- [ ] 账号基础设置（头像、简介、背景）
- [ ] 内容规划（选题库、发布计划）
- [ ] 创作流程（从选题到发布的SOP）
- [ ] 运营策略（发布时间、互动节奏）
- [ ] 数据追踪（关键指标、优化方法）

**输出**：`action_checklist.md`
