## 下载方法

### 方法1：小红书去水印工具（推荐）
1. 打开小红书 App 或网页版
2. 找到目标视频 → 点击分享 → 复制链接
3. 访问去水印网站（如：https://www.xs-download.com/）
4. 粘贴链接 → 下载视频
5. 重命名文件：`作者名_视频标题.mp4`

### 方法2：浏览器插件
1. 安装 Chrome 插件："小红书视频下载"
2. 在小红书网页版播放视频
3. 点击插件图标下载

### 方法3：手机端保存
1. 打开小红书 App
2. 找到视频 → 保存到本地
3. 通过隔空投送/微信传到电脑
4. 移动到对应目录

## metadata.json 格式

每个视频目录下创建 `metadata.json`：
```json
{
  "videos": [
    {
      "filename": "video_001.mp4",
      "title": "视频标题",
      "author": "博主名",
      "url": "原始链接",
      "download_time": "2026-02-26",
      "duration": 180,
      "likes": 12000,
      "notes": "可选备注"
    }
  ]
}
```
